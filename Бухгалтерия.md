## Заработная плата

### Установка расценок за выполнение работ

Документ предназначен для утверждения нормативов стоимости за выполнение различных видов землеустроительных и кадастровых работ. Это внутренний организационно-распорядительный документ, который служит основанием для расчета оплаты за выполнение работ. Документ расположен в подсистеме Бухгалтерия -> Заработная плата -> Установка расценок за выполнение работ.

Расценки устанавливаются на основании нормативного документа, указанного в поле **«Основание»**, с указанием наименования документа, его номера и даты.

![Screenshot_157](assets/Screenshot_157.png)

Поле **"Значение для заполнения"** является ключевым инструментом для массового установления одинаковых расценок на виды работ в одном документе. Его основное назначение - служить источником значения по умолчанию для автоматического заполнения табличной части.

![Screenshot_158](assets/Screenshot_158.png)

**Добавление видов работ и установка расценок**

В системе организовано 2 способа добавления видов работ: групповой и точечный.

**Групповой способ**

Этот метод предназначен для массового добавления видов работ, сгруппированных по тематическим разделам.

Перейдите на вкладку "Отбор для заполнения"

![Screenshot_159](assets/Screenshot_159.png)

В левой части окна выберите группу работ или несколько групп по кнопке Добавить.

В правой части окна выберите папки с необходимыми видами работ из справочника Виды работ. После заполнения условий отбора перейдите на вкладку Виды работ, убедитесь, что в шапке документа заполнено поле Значение для заполнения и нажмите кнопку **Заполнить**. Таблица с видами работ будет заполнена автоматически. После заполнения таблицы рекомендуется визуально проверить, что не осталось пустых или некорректно заполненных ячеек.

![Screenshot_160](assets/Screenshot_160.png)

**Точечный способ**

Данный метод применяется когда необходимо выборочно добавить конкретные виды работ без привязки к их групповой принадлежности.

Для этого находясь на вкладке "Виды работ", нажмите кнопку "Добавить"

![Screenshot_161](assets/Screenshot_161.png)

В новой строке подберите конкретный вид работ из справочника, можно воспользоваться поиском по наименованию или коду работы. Подберите все необходимые работы, в колонке **Норматив** установите значение стоимости для каждой строки.

![Screenshot_162](assets/Screenshot_162.png)

Для фиксации дополнительной информации о документе на вкладке Дополнительно можно прописать комментарий.

![Screenshot_163](assets/Screenshot_163.png)

Данный комментарий будет отображаться в списке всех документов "Установка расценок за выполнение работ".

![Screenshot_164](assets/Screenshot_164.png)

После заполнения документа нажмите на кнопку **Провести и закрыть**.

### Планы выполнения работ

Документ "Планы выполнения работ" представляет собой инструмент для установки индивидуальных плановых заданий для сотрудников различных должностей и подразделений. Документ расположен а подсистеме Бухгалтерия -> Заработная плата -> Планы выполнения работ.

Новый документ создается по кнопке **Создать.** Дата документа определяет период, с которого начинают действовать установленные планы. По умолчанию устанавливается текущая дата системы, с возможностью изменения, например новый план действует с 01.12.2025, тогда дату документа необходимо сменить на дату начала действия нового плана.

![Screenshot_166](assets/Screenshot_166.png)

В шапке документа необходимо заполнить реквизиты: 

- **Организация** - поле является обязательным и служит для идентификации юридического лица, для сотрудников которого устанавливаются плановые показатели.
- **Подразделение** - поле определяет структурное подразделение организации, для которого устанавливаются планы.

![Screenshot_165](assets/Screenshot_165.png)

На вкладке **Планы (индивидуальные задания)** подбираются должности для которых будет установлен план.

Нажмите кнопку "**Добавить**" в табличной части, в новой строке подберите Должность из справочника должностей.

![Screenshot_167](assets/Screenshot_167.png)

Для каждой должности в табличной части устанавливается плановый показатель выполнения работ. Для этого в строке соответствующей должности найдите колонку "План" и введите числовое значение планового показателя.

![Screenshot_168](assets/Screenshot_168.png)

На вкладке Дополнительно в поле Комментарий можно зафиксировать дополнительную информацию о документе.

После завершения работы с документом нажмите кнопку **Провести и закрыть**.

### Данные об отработанном времени

Дата документа указывает на месяц, за который ведется учет рабочего времени. Документ создается фиксации фактических данных отработанного времени за месяц.

![Screenshot_170](assets/Screenshot_170.png)

В шапке документа необходимо заполнить реквизиты: 

- **Организация** - поле является обязательным реквизитом и служит для идентификации юридического лица, в рамках которого фиксируется отработанное временя.
- **Подразделение** - поле определяет структурное подразделение организации, для сотрудников которого фиксируется отработанное временя.

![Screenshot_169](assets/Screenshot_169.png)

На вкладке Состав указывается список сотрудников. По кнопке **Заполнить** таблица будет заполнена автоматически включив всех сотрудников выбранного подразделения.

![Screenshot_171](assets/Screenshot_171.png)

Кнопка **Добавить** позволяет включить в таблицу конкретного сотрудника. Нажмите кнопку "Добавить" в табличной части документа и выберите из справочника необходимого сотрудника.

Колонка **Часы план** заполняется автоматически на основе производственного календаря. Система определяет норму рабочего времени за период с учетом праздничных и выходных дней.

Колонка **Часы факт** заполняется автоматически в соответствии с планом в возможностью корректировать данные. Для каждого сотрудника вручную откорректируйте фактически отработанное время в поле Часы факт.

![Screenshot_172](assets/Screenshot_172.png)

На вкладке Дополнительно в поле Комментарий можно зафиксировать дополнительную информацию о документе.

После завершения работы с документом нажмите кнопку **Провести и закрыть**.

### Начисление заработной платы

Документ **"Начисление заработной платы"** — это основной инструмент для расчета заработной платы сотрудников на основе выполненных работ. Документ находится в подсистеме Бухгалтерия -> Заработная плата -> Начисление заработной платы.

Новый документ создается по кнопке **Создать.** В новом документе в шапке заполните реквизиты Организация и Подразделение.

![Screenshot_186](assets/Screenshot_186.png)

Для заполнения данных нажмите на кнопку **Заполнить по нарядам**.

![Screenshot_187](assets/Screenshot_187.png)

Система автоматически формирует табличные части. В верхней таблице отображаются исполнителе с плановыми и фактическими показателями. Таблица содержит информацию о:

- **Исполнитель** - ФИО сотрудника;
- **Сумма наряд**а - общая стоимость выполненных работ;
- **Сумма начислено** - рассчитанная заработная плата к выплате;
- **Плановое количество часов** - норма часов по производственному календарю;
- **Фактическое количество часов** - отработанное время по документу Данные об отработанном времени;
- **Индивидуальное плановое задание** - установленный плановый показатель;
- **Скорректированное плановое задание** - план с учетом фактически отработанного времени;  
- **Процент выполнения плана** - рассчитанное значение (Сумма наряда / Скорректированное задание) × 100%.        

В нижней таблице формируется расшифровка со всеми выполненными видами работ и стоимостями. Таблица содержит информацию о:

- **Исполнитель** - ФИО сотрудника;
- **Вид работ** - конкретный вид выполненной работы;
- **Количество** - объем выполненной работы;
- **Сумма наряда** - стоимость работы по установленным расценкам;
- **Процент от стоимости** - доля исполнителя в стоимости работы;
- **Сумма начислено** - заработок по конкретной работе.

![Screenshot_188](assets/Screenshot_188.png)

Из документа можно распечатать печатную форму **Расчет сдельной заработной платы** по советующей кнопке.

![Screenshot_189](assets/Screenshot_189.png)

После завершения работы с документом нажмите кнопку **Провести и закрыть**.

### Сводная ведомость выполненных работ по исполнителю

Рабочее место **Сводные ведомости выполненных работ по исполнителю** представляют собой комплексную систему, предназначенную для консолидации и систематизации всех видов работ, выполненных конкретным специалистом за установленный отчетный период. 

Рабочее место расположено в подсистеме Бухгалтерия -> Заработная плата -> Сводные ведомости выполненных работ по исполнителю. Интерфейс системы представлен в виде детализированной таблицы, каждая строка которой соответствует отдельной сформированной ведомости.

![Screenshot_173](assets/Screenshot_173.png)

рабочее место открывается под текущим пользователем и в текущем периоде (текущий месяц). Для запуска формирования сводных ведомостей необходимо удостовериться, что установлен необходимый период формирования ведомости и нажать на кнопку **Сформировать автоматически**.

![Screenshot_174](assets/Screenshot_174.png)

По завершении процесса в интерфейсе списка ведомостей появляются новые строки с созданными документами.

Для удобства поиска в рабочем месте реализованы фильтры:

![Screenshot_175](assets/Screenshot_175.png)

- **Тип клиента** - категория клиента (Физическое лицо, Юридическое лицо);
- **Тип ведомости** - категория работ.

Сам документ ведомости представляет собой таблицу и содержит детализированную информацию по каждому выполненному заказу.

![Screenshot_176](assets/Screenshot_176.png)

Прежде чем инициировать процесс подписания, исполнителю необходимо тщательную проверить документ, чтобы исключить возможные ошибки.

После успешного завершения этапа проверки необходимо подписать документ электронной цифровой подписью, для этого в документе нажать на кнопку **Подписать документ**.

## Расчеты

### Ведомость по расчётам с контрагента

В системе можно сформировать отчет "Ведомость по расчётам с контрагента". **Ведомость по расчетам с клиентами** — это аналитический отчет, который предоставляет детализированную информацию о состоянии взаиморасчетов с контрагентами.  Отчет расположен в Бухгалтерия -> Отчеты -> Ведомость по расчётам с контрагента.

Отчет позволяет фильтровать данные для анализа по:

![Screenshot_190](assets/Screenshot_190.png)

- **Период** — даты, за которые анализируются расчеты;

- **Организация** — выбор юридического лица;
- **Тип клиента** — фильтр по категориям контрагентов;
- **Клиент** — конкретный контрагент для детального анализа;
- **Заказ клиента** — выбор конкретного заказа для анализа.

После изменения фильтров нажмите кнопку **Сформировать**, отчет сформируется по новым критериям.

![Screenshot_191](assets/Screenshot_191.png)

Отчет строится по многоуровневой системе группировки:

- **Уровень 1: Клиент**
- **Уровень 2: Заказ клиента**
- **Уровень 3: Документы-основания**

### **Ак**т сверки

Чтобы сформировать акт сверки с контрагентами, необходимо зайти в раздел Бухгалтерия -> Расчеты -> Акт сверки.

В журнале документов необходимо нажать на кнопку «Создать». После этого откроется документ, который необходимо будет заполнить следующим образом:

- **Организация:** необходимо указать вашу организацию;
- **Контрагент:** контрагента (с которым необходима сверка);
- **Период** (с какого по какое число необходима сверка);
- **Договор:** если по определенному договору, то указываем, в других случаях – пусто.

После того, как форма документа заполнена, на вкладке По данным организации необходимо нажать на кнопку «Заполнить»

![Screenshot_208](assets/Screenshot_208.png)

После этого все документы расчетов с данным контрагентом подтянутся в табличную часть документа. Таким же образом заполняется вкладка «По данным контрагента».

![Screenshot_209](assets/Screenshot_209.png)

Чтобы вывести на печать акт сверки, необходимо записать документ, а после нажать на кнопку «Акт сверки».

![Screenshot_210](assets/Screenshot_210.png)

### Приходный ордер

Операции по поступлению в организацию материалов от поставщиков регистрируются с помощью документа Приходный ордер. Документ расположен в Бухгалтерия -> ТМЦ -> Приходный ордер.

В журнале документов необходимо нажать на кнопку «Создать». После этого откроется документ, который необходимо будет заполнить следующим образом:

- **Организация:** необходимо указать организацию куда приходит номенклатур;
- **Поставщик:** контрагент поставляющий номенклатуру;
- **Склад:** склад на который приходит номенклатура.

Для заполнения номенклатуры в табличной части нажмите кнопку **Добавить**.

![Screenshot_211](assets/Screenshot_211.png)

В новой строке укажите номенклатуру, количество и цену.

После завершения работы с документом нажмите кнопку **Провести и закрыть**.
